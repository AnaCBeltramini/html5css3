Meta tags

Meta tags são linhas de códigos que descrevem o conteúdo do site (SEO)

viewport

Meta tag description instrui o computador como controlar o tamanho e o dimencionamento da página

charset

A meta tag charset é usada para indicar o padrão de codificação de caracteres

description

A meta tag description descreve a página para os buscadores(SEO) máximo 156 caracteres
